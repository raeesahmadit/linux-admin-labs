TOP(table of processes) to see real-time processes and resources being used like Taskmanager in windows

1. Top :    refresh in 3 sec
2. most use for monitoring CPU and Memory and complete detailed per process

top > c = show cmd absolute path
top > k = kill a process by PID
top > n = to change the number of task displayed
top > d,s = to change interval of refresh
top > M = to sort running process by Memory usage
top > r = you can change the nice values of a PID
top > u = to filter task by user
top > f = field management > remove starts which dont want to see
top > h = help
top > z,b = toggle color/mono; bold/reverse
top > x,y = x sort field; y running task
top > i = process which some memory usage
