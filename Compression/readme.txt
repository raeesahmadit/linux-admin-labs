1. to compress a file use gzip     : gzip file1
   to uncompress a file use gunzip : gunzip file1
  
   we do archieve to gather all files in one place

2. to archieve a directory use tar :
                tar -cvf humans.tar humans
   to archieve and compress at a same time:
               tar -czvf humans.tar.gz humans
    to uncompress a directory: 
               tar -xzvf humans.tar.gz folder-name/

3. to zip a file : zip humans.zip humans
    to unzip a file : unzip humans.zip

    but tar is more efficent because it gives more details 
    but zip dont.
